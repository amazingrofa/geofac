# Documentation

This directory contains the documentation for the `geofac` project, organized into the following categories:

*   **[Theory](./theory/)**: Foundational concepts, whitepapers, and core principles of geometric resonance factorization.
*   **[Implementation](./implementation/)d/)**: Code architecture, implementation details, and style guides.
*   **[Validation](./validation/)**: Verification methods, testing, and proof packs.
*   **[Project](./project/)**: Project management, changelogs, and administrative documents. See the original [README](./project/README.md) for more details.
*   **[Archive](./archive/)**: Potentially outdated or experimental work.

Please explore the directories to find the information you need.
